# Machine Learning in Python

This repository contains Python programs and practice files related to machine learning, core libraries, preprocessing, model building, evaluation, visualization, and foundational concepts.

The goal of this repository is to build a strong understanding of machine learning step by step through clean and practical Python implementations.

Topics included in this repository:
- machine learning basics
- NumPy
- Pandas
- Matplotlib
- Seaborn
- scikit-learn
- preprocessing
- model evaluation
- classification
- regression
- clustering
- foundational ML workflows
